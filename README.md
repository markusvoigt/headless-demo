This project uses [Next.js](https://nextjs.org/) together with [TailwindCSS](https://tailwindcss.com) to build a tiny e-commerce storefront on top of Shopiy. 

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
